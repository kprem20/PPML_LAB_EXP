## 💻 How to Run

1. Install Python (version 3.x recommended)
2. Clone this repository:

   ```
   git clone https://github.com/kprem20/PPML_LAB_EXP
   ```
3. Open the folder and run any file:

   ```
   python filename.py
   ```
